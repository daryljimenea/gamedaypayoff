Have Book/Player, Want Other

If you only have the GameDay Payoff - Book application or the GameDay Payoff - Player application and desire to have both, please double-click on the "gamedaypayoffinstaller.exe".  You might either have this file already or you can download it from GameDay Payoff.  Upon opening the file, you will choose the Modify option in the installer.  Then check the application you want to have and proceed with the installation.  It's that easy!
