Restoring Database from a Backup

Restoring the database will return all your wagers and settings to exactly where they were when you made the backup.  That includes everything you need to be up and running again.  Obviously, you must have already made a backup to be able to restore it.  See the topic "Backing Up the Database" in the Help section for instructions on how to do so.

There are different databases for the Book and Player.  To restore the database, do the following:

Restoring onto same computer:
1) Exit the application
2) Go to your Windows search feature located when you hit "start" in the lower left hand of your screen and find the file "BOOK.FDB" for Book application or "PLAYER.FDB" for Player application.  Make sure that you option to search in hidden folders and files.  If you have 2 of the same, make sure that you always apply these instructions only to the one that is located in the "GameDayPayoffData" folder.
3) Once the file is located, replace that file with the one from your source.  
4) Launch the application
5) Validate the subscription by entering the image text.  If it does not "take", please enter again keeping in mind that an "O" can look like a "0" and an "l" can look like a "1".  Try different like characters to make it work.

Restoring onto different computer that has your application (Book or Player):
1) Exit the application
2) Go to your Windows search feature located when you hit "start" in the lower left hand of your screen and find the file "BOOK.FDB" for Book application or "PLAYER.FDB" for Player application.  Make sure that you option to search in hidden folders and files.  If you have 2 of the same, make sure that you always apply these instructions only to the one that is located in the "GameDayPayoffData" folder.
3) Once the file is located, replace that file with the one from your source.  REPLACING THE FILE WILL EFFECTIVELY DELETE THE USER'S AND SETTINGS FOR THE APPLICATION ON THAT COMPUTER.  
4) Launch the application
5) Validate the subscription by entering the image text.  If it does not "take", please enter again keeping in mind that an "O" can look like a "0" and an "l" can look like a "1".  Try different like characters to make it work.

Restoring onto different computer that does not have your application (Book or Player):
1) Install the application you desire and then exit the application
2) Go to your Windows search feature located when you hit "start" in the lower left hand of your screen and find the file "BOOK.FDB" for Book application or "PLAYER.FDB" for Player application.  Make sure that you option to search in hidden folders and files.  If you have 2 of the same, make sure that you always apply these instructions only to the one that is located in the "GameDayPayoffData" folder.
3) Once the file is located, replace that file with the one from your source.
4) Launch the application
5) Validate the subscription by entering the image text.  If it does not "take", please enter again keeping in mind that an "O" can look like a "0" and an "l" can look like a "1".  Try different like characters to make it work.
