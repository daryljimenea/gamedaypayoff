Transferring to Another Computer

With one exception, you can move entire databases from one computer to another.  However, you cannot move just one Player from one GameDay Payoff application to another GameDay Payoff application.  The entire database (with ALL Players) needs to move or nothing.

The one exception to moving the entire database is that if you installed both applications on the same computer and now need them on separate computers, it will not work.  You will need to do a full uninstall (see topic "Uninstall Book/Player" in Help) and start from the beginning for one of the applications.  You can take your pick on which one you would like to remove.  However, if you do find yourself needing to transfer both of the applications to another computer, that can easily be accomplished.

All you need to do is move the database from the computer you are on, to the computer you want to use.  However, copying the same database and running it on two different computers at the same time will not work.  Neither can you merge databases if you have been working on two different computers and wish to combine the data.  If, after following the instructions below, your application does not accept the transfer based on the subscription, please see "Validate Subscription" in the Help section.

To transfer all of your settings (including subscription, direct connections, wagers, policy, etc.) to a new computer, do as follows:
1) On the computer you are transferring from, exit the application.
2) Go to your Windows search feature located when you hit "start" in the lower left hand of your screen and find the file "BOOK.FDB" for Book application or "PLAYER.FDB" for Player application.  These are the associated databases for each.  Make sure that you option to search in hidden folders and files or you will not find it.  If you have 2 of the same, make sure that you always apply these instructions only to the one that is located in the "GameDayPayoffData" folder.
3) Copy this file onto a medium source such as a CD, DVD, External Hard Drive, Flash Drive, or any other like product.  Before copying, make sure that the computer you are transferring to is capable of reading from that medium source so that you may transfer the database.
4) On the computer that you are transferring the database to, install the Book application or Player application.
5) Exit the application.  You do not need to wait for the feed to update because you will not be using the database that the feed is being written onto.
6) Repeat Step 2 and then jump to step 7 upon concluding this.
7) Once the file is located on the computer you are transferring the database to, replace that file with the one from your medium source that has the database you wish to use.  To do this, you can just copy from the medium source and paste onto the computer you are transferring to.  When it prompts you to replace, hit Ok.
8) Launch the application on the computer that you transferred the database to.
9) Validate the subscription by entering the image text.  If it does not "take", please enter again keeping in mind that an "O" can look like a "0" and an "l" can look like a "1".  Try different like characters to make it work.  If you cannot figure it out, just exit the application, re-open it, and then try again.  Once you validate the subscription, you will notice that all of your settings and accounts are there.
