Backing Up the Database

Backing up the Database is always a good idea.  The database contains all of your wager information and settings.  That includes everything you need to be up and running again.  The more often you backup the database, the better, but it is not a requirement.  There are different databases for the Book and Player.  To back up your database for either the Book or Player, just do as follows:

1) Exit the application
2) Go to your Windows search feature located when you hit "start" in the lower left hand of your screen and find the file "BOOK.FDB" for Book application or "PLAYER.FDB" for Player application.  Make sure that you option to search in hidden folders and files.  If you have 2 of the same, make sure that you always apply these instructions only to the one that is located in the "GameDayPayoffData" folder.
3) Copy this file onto your desired medium source such as a CD, DVD, External Hard Drive, Flash Drive, main drive, or any other like product.  It is recommended that you do not back the database onto the same computer as the original in case something physically happens to that computer.  If you do have a backup and something happens to your computer, you will have everything you need.

To restore that database, see the topic "Restoring Database from a Backup" in the Help section.
