Log Out

When you Log Out from the application, you close your account, but the application remains running unless you exit.  This way, you can have the application up and running without worrying about anyone having access.  To completely close the application, use the Exit under the File menu or the Close box in the upper right hand corner of the application.  Closing the application also serves as logging the user out.  Meaning, whenever the application is launched, password protected users will have to re-enter their password to log in.
