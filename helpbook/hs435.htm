Line Monitor Tips

The custom OTB feature is a pretty powerful feature that allows you to do a couple of things.
1) It allows you to take a wagerable off the board (OTB) if you do not want to offer it at all.
2) It allows you to override the default OTB for a particular wagerable so that it never goes OTB.
3) It allows you to make a wagerable available again after the default OTB has taken it of the board.  This, in theory, would be done after you move the line to promote action on the other side of the wagerable.



General Tips:
1) To use Custom OTB for a wagerable, just go to the box underneath the checkbox for that wagerable, enter an amount, and then tab away from that box.
2) A dark red box always indicates that you are using a Custom OTB value and that you are overriding the default OTB value.
3) To take a wagerable off of Custom OTB and have it revert back to the default OTB, just go into the Custom OTB box for the desired wagerable, hit delete, and tab out of it.
4) If you do not want any wagerables going OTB for any reason at any time, just set the Default OTB field to a huge dollar amount so that it will never be reached.  For a particular wagerable, you can always make it smaller than this amount if you do wish for it to go OTB for any given amount.
