Uninstall, Modify, or Repair Book/Player

You can uninstall GameDay Payoff one of three ways.  You can either use the "gamedaypayoffinstaller.exe", you can go to the Add or Remove Programs from the control panel, or go to "Start" in lower left corner, "All Programs", and then the GameDay Payoff folder where there is an uninstall.  Any of the three methods are equally effective.  

Before uninstalling, it is first recommended that you close both the Book and Player applications before proceeding.  Once in the uninstaller, you have the following options.

1a)  Modify (Player or Book is already installed):  Use the modify option if you want to install the Player if you have only the Book OR the Book if you have only the Player.
1b)  Modify (Both Player and Book are installed):  Use the modify option if you are just uninstalling either the Player or the Book GameDay Payoff application.
2)  Repair:  Use the repair option if you are having trouble with the application, but would like to keep all of your settings and data.
3)  Remove:  Use this option if you would like to completely remove GameDay Payoff in its entirety.  That means both the Book and the Player applications will be removed and all files related to GameDay Payoff.

If you should decide to reinstall at a later date, your data and settings will still be there unless you do a full uninstall.

FULL UNINSTALL (includes deleting .FDB file):
After doing an uninstall, go to your Windows search feature located when you hit "start" in the lower left hand of your screen.  Then choose "All files and folders".  Then choose "More advanced options" and check "Search hidden files and folders".  Then in "All or part of the file name:" at the top, put in "BOOK.FDB" for Book application or "PLAYER.FDB" for Player application and hit Search.  If you have 2 of the same, make sure that you always apply these instructions only to the one that is located in the "GameDayPayoffData" folder.  Delete this file and you will forever lose your data and settings.  Doing a Full Uninstall is not reversible, but should you decide to continue using GameDay Payoff after you uninstall it, you may download another installer exe from the website and start completely over.  DOING A FULL UNINSTALL WILL NOT GET YOU AROUND PAYING.
