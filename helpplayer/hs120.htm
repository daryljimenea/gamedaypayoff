Laptop Computers

GameDay Payoff - Player version will operate correctly anywhere there is an internet connection, so if you are using a laptop, you may use it from anywhere the internet is available.  Other than having an internet connection, there is nothing required for a Player to do for GameDay Payoff - Player to correctly operate.

If you are using a laptop for mobility, GameDay Payoff - Book will operate correctly anytime there is an internet connection and you can receive a signal from your Players (if you have Direct Connection Players).  For more on this, see "Direct Connection Books Must Receive Signal" in the Help.
